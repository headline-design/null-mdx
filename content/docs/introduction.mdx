## Quick Start

To get started, you'll need to:

1. Install the dependencies
2. Configure your environment
3. Run the development server

\`\`\`bash
npm install
npm run dev
\`\`\`

<Callout type="info">
  Make sure you have Node.js 18+ installed before proceeding.
</Callout>
